# Scientific Calculator - JavaFX

A desktop scientific calculator built with Java and JavaFX.  
This application supports both basic arithmetic operations and scientific functions such as trigonometric functions, logarithms, square root, and squaring.

## Features
 Basic operations: addition, subtraction, multiplication, division
 Scientific functions: sin, cos, tan, log, ln, sqrt, x²
 Graphical user interface built with JavaFX
 CSS styling for a clean modern look
 Error handling for invalid operations

1.	Install Java JDK
2.	Install JavaFX SDK
3.	Configure JavaFX in VS Code
4.	Run Main.java
